Detailed Description

Loads and processes images for menu items.

The ImageLoader class loads images for menu items and applies processing based on the number of levels unlocked. It emits a signal when the image is ready.

Starts the image loading and processing.

This function loads the base image, applies grayscale conversion if the index is greater than or equal to the number of levels unlocked, scales the images to a fixed size, overlays a frame, and emits the imageReady signal.
