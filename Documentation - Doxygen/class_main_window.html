◆ handleMenuFinished()

void MainWindow::handleMenuFinished ( )

Handles the finished event of the menu.

The function is called when the menu has finished, indicating that the game should start. It initializes the TileSet and MapSection objects, creates a vector of TileSet pointers, and loads the map. It also creates a Level object with the specified level number and starting position, sets the background image of the level's scene, and starts the level.
