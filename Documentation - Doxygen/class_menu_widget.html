◆ animateSelection()

void MenuWidget::animateSelection ( )

Animates the selection of a frame in the menu.

This function animates the selection of a frame in the menu. It calculates the desired position below the titleView, and then applies animations to the frames to shrink and fade out the unselected frames, move the selected frame to the desired position, and add a glowing effect to the selected frame. Additionally, it applies a fading animation to the entire MenuWidget after a certain delay, using a QGraphicsOpacityEffect. The function also connects the 'finished' signal of the fading animation to the 'finished' signal of the MenuWidget.

◆ moveCursorToSelectedFrame()

void MenuWidget::moveCursorToSelectedFrame ( )

Moves the cursor to the selected frame in the menu.

This function calculates the global position of the center left of the selected frame, loads a cursor image, creates a QCursor object with the image and hot spot, and sets the cursor to the created QCursor. Finally, it moves the mouse pointer to the calculated position.

◆ saga()

void MenuWidget::saga ( )

Starts the saga menu animation.

This function initiates the saga menu animation. It plays the "saga.mp3" audio file and sets the "saga.jpg" image as the background. If the "saga.mp3" finishes playing, it sets the "bgstart.gif" image as the background, creates QLabel and QMovie objects for the gif animation, displays a logo with a custom font, creates QPushButton objects for the play and settings buttons with custom stylesheets, and shows cursor animations. The function connects the clicked() signals of the playButton and settingsButton to corresponding slots. After the animation, it sets the "menubg.png" image as the background and calls the initMenu() function to initialize the menu.
